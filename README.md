## About
MarkMyDay is a comprehensive mobile application built from scratch to streamline school operations. It provides a unified platform for tracking attendance, managing communications, and organizing daily school activities.

## Core Features
* **Role-Based Access:** Secure authentication and distinct interfaces tailored for Admins, Teachers, and Students.
* **Attendance Management:** Quick, reliable daily attendance tracking and automated syncing.
* **School Updates:** A centralized hub for school-wide announcements and news.
* **In-App Notifications:** Real-time push alerts for important events, schedule changes, and targeted messages.

## Tech Stack
* **Frontend:** Jetpack Compose (Kotlin & Java)
* **Backend:** Firebase Authentication & Cloud Firestore
* **Version Control:** Git & GitHub

## Future Roadmap
* Offline mode for marking attendance without internet access.
* PDF and CSV report generation for attendance data.
* Dedicated parent portal for real-time monitoring.
* Automated analytics for student performance insights.
